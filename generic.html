Movie Player Finder

Objective:

- Develop a movie player application in C++ that can build and manage a playlist from a large collection of movie titles, utilizing features such as classes, templated data structures, file input, and operator overloading.

Key Features:

- Movie and Playlist Management: Create and manage a collection of movies with various attributes (e.g., title, genre, rating).
- Templated Data Structure: Implement a queue or stack using C++ templates to handle various data types.
- File Input: Read movie data from files and populate the playlist.
- Operator Overloading: Customize operators to manipulate the data structure intuitively.
- Programming Language: C++

Core Components:

- Movie Class: Represents individual movies with attributes like title, genre, and rating.
- Playlist Class: Manages a collection of Movie objects.
- Templated Queue or Stack: Handles operations like push, pop, enqueue, and dequeue for various data types.
- File Handling: Reads movie data from files and ensures robust error checking.
- Operator Overloading: Overloads operators to provide intuitive access and manipulation of the data structure.

Retriever Bell

Objective:

- Develop a system to manage phone connections for RetrieverBell, a new phone company, handling both local and long-distance calls.

Key Features:

- Local Calls: Connect phones within the same area code directly through local switchboards.
- Long-Distance Calls: Route calls through connected switchboards if the phones are in different area codes.
- Programming Language: Python
- Arrays: Store phone numbers and area codes.
- Dictionaries: Map area codes to phone numbers and switchboards.

Core Components:

- Area Code Management: Maps area codes to phone numbers and switchboards.
- Local Calls: Connect directly if area codes match.
- Long-Distance Calls: Route through connected switchboards.
